# Code Review Skill

审查代码时遵循以下步骤：

## 检查清单

1. **正确性**：代码是否实现了预期功能？
2. **边界情况**：错误条件是否处理了？
3. **风格**：是否符合项目规范？
4. **性能**：是否有明显的低效实现？

## 如何给反馈

- 具体指出需要改什么
- 解释为什么，不只是说什么
- 尽可能建议替代方案
